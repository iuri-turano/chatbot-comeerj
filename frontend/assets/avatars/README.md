# Avatar do Assistente - Chico Xavier

## Instruções

Para adicionar a foto do Chico Xavier como avatar do assistente, siga os passos:

1. **Encontre uma imagem do Chico Xavier** (foto de domínio público ou com licença apropriada)

2. **Redimensione a imagem** para 40x40 pixels ou 80x80 pixels (para suportar telas retina)

3. **Salve a imagem nesta pasta** com o nome: `chico_xavier.png`

4. **Fontes recomendadas** para imagens de domínio público:
   - Wikimedia Commons
   - Arquivo público brasileiro
   - Fotos históricas em domínio público

## Formato recomendado

- **Formato:** PNG ou JPG
- **Tamanho:** 40x40px (ou 80x80px para retina)
- **Fundo:** Preferencialmente transparente (PNG)
- **Nome do arquivo:** `chico_xavier.png`

## Enquanto isso...

O sistema está configurado para usar um emoji "🤖" como placeholder. Quando você adicionar a imagem do Chico Xavier, o avatar será automaticamente atualizado.
